# launch
Solidity contracts of the smol404 project

This project is a launchpad that deploys an Agent Token: a tradable ERC404 token natively linked to a dedicated BAP (BAP-578) agent.
The token provides liquidity and distribution (bonding curve + LP migration), while the BAP proxy provides the agent’s identity, metadata (URI), and control surface. Each launch creates a verifiable one-to-one relationship between the token and its agent, enabling “tokenized agents” rather than generic memecoins.
